# Reto-Churn-Rate
* El proyecto trataba de analizar las razones por lo que una empresa de telecomunicaciones estaba perdiendo clientes
* Se reaizo una limpieza de datos y estandarización usando la liberia Pandas
* Posteriormente se realizaron gráficos para poder ver la distribución de clientes que se quedan y los que no,con respecto a las otras variables cualitativas y cuantitativas.
* Se dio un perfil del cliente que abandona la empresa y sugerencias.
